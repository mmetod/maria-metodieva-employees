Проектът работи по следния начин:
На база текстов файл във формат empID, projectID, dateFrom, dateTo
намира и извежда в конзолата:
	1.двамата служителя, които са работили най-дълго по 1 проект 
	(и проектът, по който са работили, както и колко дни са работили заедно по него)
	2.двамата служителя, които са работили най-дълго заедно (може по различни проекти)
	(и проектите, по които са работили, както и колко дни са работили заедно по тях)
	
В проекта има проверки за грешни данни в текстовия файл -
- извежда съобщение в конзолата при открити грешни данни и тях ги пропуска при изчислението

В текстовия файл датите могат да са в 5 формата:
dd.MM.yyyy, dd/MM/yyyy, yyyy-MM-dd, yyyy/MM/dd, yyyy.MM.dd
Тези формати се зачитат за верни, останалите формати и датите преди 1900г. се зачитат за грешни

Текстовият файл трябва да е поставен в поддиректория files на проекта и да е с име employees.txt
Примерен файл има в проекта.

Проектът се стартира от клас Demo (package demo)
